# The Cricket Corporate Collective — Premium B2B Website Starter

This is a production-ready starter build for a premium B2B proposal-generation website.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- App Router
- API route for proposal form submissions
- Zod validation

## Brand System

- Main background: #FFFFFF
- Section background: #F8FAFC
- Primary navy: #031E49
- Premium gold: #E5A823

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Recommended Deployment

- Vercel for hosting
- HubSpot CRM for lead capture
- Resend or Mailgun for email notifications
- Supabase/PostgreSQL if storing leads in a database
